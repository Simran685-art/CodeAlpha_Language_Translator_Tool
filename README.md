# 🌍 Advanced Language Translation Tool

## 📌 Project Description
An AI-powered Language Translation Tool developed using Python and Streamlit. The application translates text between multiple languages, detects the input language automatically, converts translated text into speech, maintains translation history, and allows users to download translations.

## 🚀 Features

- Multi-language Translation
- Auto Language Detection
- Text-to-Speech (Audio Output)
- Translation History
- Character Count
- Word Count
- Download Translation
- User-Friendly Interface
- Real-Time Translation

## 🛠️ Technologies Used

- Python
- Streamlit
- Deep Translator
- gTTS
- LangDetect

Install dependencies:

```bash

```

```bash
streamlit run app.py
```
